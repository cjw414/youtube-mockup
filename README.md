# youtube-mockup
유튜브 웹 서비스의 UI를 목업으로 구현한 학습용 미니 프로젝트

1. 화면설계서 및 결과 문서 작성
 - 화면설계서, 발표자료, 보고서를 문서화합니다.

2. 헤더/썸네일 등 핵심 UI 요소 구현
 - 주요 구성 요소를 시각적으로 구현합니다.

3. CSS를 활용한 기본 스타일링 적용
 - 레이아웃 중심으로 CSS를 적용하여 디자인합니다.

4. DOM 트리 정리 및 시맨틱 구조 개선
 - 마크업의 구조를 DOM 관점에서 정리 및 개선합니다.

5. HTML을 사용한 기본 구조 작성
 - 시맨틱 태그로 구조를 설계하고 마크업을 작성합니다.

6. 유튜브 UI 구조 분석 및 화면 구성 요소 분해
 - 실제 유튜브 화면을 분석하고 구성요소를 파악합니다.
